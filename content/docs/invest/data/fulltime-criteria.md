# 全职交易评估标准

> 全职交易的目标不是证明自己能够赚钱，而是证明自己拥有一套能够在不同市场环境下持续创造正期望值、严格控制风险并长期稳定执行的交易系统。

评估周期：2026.07 - 2029.07（三年）

## 量化指标（硬性，必须全部达标）

| 指标 | 标准 | 说明 |
|------|------|------|
| 年化收益率 | 3 年 CAGR ≥ 15% | $350k x 15% = $52.5k ≈ 覆盖年支出 7 万 SGD |
| 正收益年份 | 至少 2/3 年盈利，且任何单年亏损 < 10% | 三年累计收益需达到 CAGR ≥15% 目标；允许一年小幅亏损，但必须下一年恢复并创新高 |
| 最大回撤 | 任意时点 peak-to-trough < 20% | 超过 20% = 风控有问题 |
| 单月最大亏损 | < 5% | 能控制单月损失幅度 |
| 本金规模 | ≥ $300,000 USD | 使得 15% 收益即可覆盖开支，不需要冒大风险 |
| Sharpe Ratio | > 1 | 风险调整后收益。Sortino Ratio > 1.5 作为补充参考 |
| Expectancy | 长期为正 | 每月统计 Win Rate × Avg Win - Loss Rate × Avg Loss > 0 |
| 交易样本 | 三年 ≥ 300 笔 | 样本不足则统计无意义，可根据实际策略频率调整 |
| Recovery Time | 稳定 | 每次 >10% 回撤，恢复到 Equity High 的时间需有记录且无恶化趋势 |

## 定性条件（软性，综合评估）

| 条件 | 标准 |
|------|------|
| 规则执行率 | > 90% 的交易遵循 Trade Checklist |
| 情绪管理 | 不再出现报复性交易、FOMO 驱动的冲动操作 |
| 市场适应性 | 至少经历一轮明显回调/震荡市，仍能正收益或控制亏损 |
| 收入独立性 | 有工作或其他收入覆盖生活开支，投资收益是增量 |
| 系统内化 | 规则不需要刻意提醒，已成为交易本能 |
| 记录完整性 | 每笔交易有记录，每月有 track record 更新 |

## Market Regime 验证

必须统计并记录以下三种市场环境的表现：

| 市场环境 | 定义 | 目标 |
|----------|------|------|
| Bull Market | 主要指数处于上升趋势（>MA200） | 跑赢基准或获得绝对正收益 |
| Bear Market | 主要指数从高点回撤 >20% | 控制亏损 <10%，或正收益 |
| Sideway Market | 主要指数横盘震荡（无明确趋势） | 正收益或小幅亏损 <5% |

目标：证明系统不是只适用于某一种市场环境。

## 三年阶段目标

### Year 1: 2026.07 - 2027.07（规则建立 + 执行磨合期）
- 目标：正收益，回撤 < 15%
- 重点：建立并内化交易规则，提升执行率
- 允许：偶尔的情绪性操作（但必须记录和复盘）

### Year 2: 2027.07 - 2028.07（系统验证期）
- 目标：≥ 15% 收益，经历不同市场环境
- 重点：验证规则在不同市况下的适应性
- 期望：报复性交易和 FOMO 操作基本消失

### Year 3: 2028.07 - 2029.07（稳定输出期）
- 目标：≥ 15% 收益，情绪和执行完全内化
- 重点：稳定性和可重复性
- 期望：交易像呼吸一样自然，不需要和情绪对抗

## 2029 年中决策树

```
3 年 CAGR ≥ 15%？
  AND 至少 2/3 年盈利，单年亏损 < 10%？
  AND 最大回撤 < 20%？
  AND 本金 ≥ $300k？
  AND 规则执行率 > 90%？
  AND Sharpe > 1？
  AND Expectancy 长期为正？
  AND 经历过 Bull / Bear / Sideway 且表现稳定？
  AND Recovery Time 无恶化趋势？
  AND 无重大情绪失控事件（Year 3）？

  最后验证：
  收益是否来自：
    ✓ 可重复的系统逻辑
    ✓ 可解释的交易决策
    ✓ 可持续执行的规则框架

  如果收益主要来自极少数异常交易 → 系统未成熟，不达标

  → 全部 Yes → 可以认真考虑全职（建议先半职过渡 6 个月）
  → 任一 No → 继续工作 + 投资，延长评估期
```

## 系统稳定性（System Stability）

最终决定是否全职，不是收益，而是系统是否已经稳定。

| 维度 | 标准 |
|------|------|
| Rule Execution | > 90% |
| Expectancy | 长期为正 |
| Sharpe Ratio | > 1 |
| Recovery Time | 稳定，无恶化趋势 |
| Market Regime | 经历 Bull / Bear / Sideway，均有记录和合理表现 |
| 执行一致性 | 三年以上保持一致执行，无显著波动 |
| 收益来源 | 可重复、可解释、可持续 — 非极少数异常交易驱动 |

## 重要提醒

- 这个标准是给三年后的自己看的，现在不需要焦虑是否达标
- 当前阶段的任务是：找到工作 + 打磨规则 + 如实记录
- 50% YTD 不代表以后每年都能做到，牛市放大了所有人的收益
- TDOC/SE 的教训：一两年的好成绩不能代表系统稳定
- 全职交易不是目标，而是能力到了以后的一个选项
- 年化 15% 不改高 — 管理自有资金、控制回撤 <20%、不依赖高杠杆，长期 15% 已经非常优秀且现实
- Rules > Emotion > Prediction：规则重于情绪，系统重于预测
