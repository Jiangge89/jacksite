## Calculation Method

- Return % = (End Equity - Start Equity - Net Inflows) / Start Equity
- Max Drawdown = peak-to-trough 最大回撤（月内或月间）
- Rule Adherence = 遵循 checklist 的交易笔数 / 总交易笔数
