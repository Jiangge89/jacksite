## 为什么关注

利率、就业、通胀直接影响成长股估值。宏观是背景板——不用预测方向，但必须知道当前在哪。

## 逻辑链

| 组合 | 对成长股 |
|------|----------|
| 弱就业 + 高通胀 | 最不利（滞胀） |
| 弱就业 + 通胀回落 | 受益（降息预期） |
| 强就业 + 高通胀 | 估值承压 |
| 强就业 + 通胀回落 | 最佳（软着陆） |

## 操作含义

- 不要预测 Fed，跟随数据
- 宏观提供仓位管理信号，不提供选股信号
- 产业逻辑 > 宏观逻辑
