## Position Sizing

基准：Total Equity（现金 + 持仓市值），每周一更新。

| 级别 | 占比 |
|------|------|
| Starter | 2% |
| Half | 5% |
| Full | 10% |
| Hard Cap | 15% |
| 板块上限 | 25% |

- 浮亏 >10%：冻结加仓，重新评估 thesis
- 浮亏 >20%：减仓至 2% 或清仓

## Add Rules

只在盈利仓位上加仓。Never average down simply because price fell.

- Starter → Half：浮盈 >5% + 趋势确认 + 量确认
- Half → Full：浮盈 >10% + 催化剂验证 thesis

## Exit Rules

- 接近阻力 / 超限 → 减 1/4 到 1/3
- Thesis 变弱 → 减到 Starter (2%)
- Thesis 破坏 / 浮亏 >20% → 清仓
- 中线退出用 thesis + 趋势结构（HH/HL），不用固定百分比止损

## Earnings Rules

- 超过 5% 的仓位：财报前必须决定持有/减仓/对冲
- 超过 10% 的仓位：财报前必须减到 10% 以内
- 财报前 3 天内不建新仓

## Options Rules

- Maximum exposure：5%（按权利金成本计）
- Never average down options
- 到期前 14 天必须决定 roll / close / hold
- 学完期权基础前不开新仓

## Emotion Rules

Never trade because of：FOMO、Fear、Need to recover losses、Anchoring、Dip-buying impulse

Always wait 30 minutes before large trades.
